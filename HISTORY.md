# 更新履歴

## Version 1.0.0 (2025-08-11)

### 🎉 初回リリース

DroidCam Stream Deck Pluginの最初の公式リリースです。スマートフォンのカメラをWebカメラとして使用できるDroidCamアプリを、Stream Deckから完全に制御できるようになりました。

### ✨ 主要機能

#### 📱 接続管理
- **接続状態モニター**: DroidCamへの接続をリアルタイムで監視
- **バッテリー状態表示**: デバイスのバッテリー残量と充電状態を表示
- **自動再接続**: 接続が切断された場合の自動復旧

#### 📸 基本カメラ制御
- **カメラ切り替え**: 前面/背面カメラをワンタッチで切り替え
- **LEDフラッシュ制御**: カメラのLEDライトをON/OFF
- **マイク制御**: マイクのミュート/解除をボタンで操作

#### 🎯 高度なフォーカス制御
- **オートフォーカス実行**: 手動でAFをトリガー
- **フォーカスモード切り替え**: Normal/Macro/Continuous/Infinityの4モード対応
- **マニュアルフォーカス**: 細かいフォーカス調整が可能

#### 🎨 ホワイトバランス制御
- **9種類のWBプリセット**: Auto/白熱灯/蛍光灯/昼光/曇り/夕暮れ/日陰など
- **手動WB調整**: 0-100の範囲で微調整可能
- **WBロック機能**: 現在の色温度を固定

#### 📊 露出・ズーム制御（Pro版限定）
- **ダイアルズーム制御**: Stream Deck+のダイアルで1.0x〜8.0xを無段階調整
- **ダイアル露出制御**: -2.0〜+2.0 EVを細かく調整
- **プリセットボタン**: よく使う値をワンタッチで切り替え
- **露出ロック**: 現在の露出設定を固定

#### ⚙️ システム制御
- **停止/再開トグル**: DroidCamアプリの一時停止と再起動
- **ステータス表示**: 動作状態を視覚的に表示

### 🌍 国際化対応
- **多言語サポート**: 日本語と英語に完全対応
- **自動言語切り替え**: Stream Deckのシステム言語に応じて自動選択
- **完全ローカライズ**: すべてのUI要素とメッセージを翻訳

### 🛠 技術仕様
- **Stream Deck SDK**: v6.4+対応
- **TypeScript**: 型安全な実装
- **モジュラー設計**: 各アクションを独立したクラスとして実装
- **エラーハンドリング**: 接続エラーやPro版制限を適切に処理

### 📋 システム要件
- Stream Deck 6.6以上
- macOS 10.15以上 または Windows 10以上
- DroidCamアプリ（スマートフォン側）
- 同一ネットワーク内での接続

### 📝 既知の制限事項
- ズーム・露出制御機能はDroidCam Pro版が必要
- 一部のAndroidデバイスでマニュアルフォーカスが使用できない場合がある
- iOSデバイスではLEDトーチが利用できない場合がある

### 🔧 インストール方法
1. [Releases](https://github.com/nantokaworks/streamdeck-droidcam/releases)から`.streamDeckPlugin`ファイルをダウンロード
2. ダブルクリックしてStream Deckにインストール
3. DroidCamアプリを起動し、IPアドレスを確認
4. Stream Deckで接続設定を行う
